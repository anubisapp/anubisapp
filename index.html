We'll be releasing the code when we believe we've hit a "stable" release - as there's some core changes we need to do first before we're happy, we're discouraging you from downloading it already.

Why make this?

The original Anubis is nice and a good start into a web monitoring UI, but it lacks many features that we deem to be extremely useful. It's also not the prettiest, so we aim to improve both of these aspects accordingly.
